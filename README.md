# Subscription Tracker Dashboard

Interactive web-based dashboard designed to centrally manage and monitor recurring service subscriptions (SaaS/tools). This application helps teams track renewal dates, calculate remaining active days in real-time, and analyze monthly expenditures efficiently.

## 🚀 Key Features

- Statistical Overview Dashboard
- Integrated CRUD Operations
- Automated Days Remaining Calculation
- Instant Search & Sorting
- Data Export (CSV)
- Persistent Local Storage

## 🛠️ Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- Vanilla JavaScript (ES6+)
- LocalStorage API

## 📂 Project Structure

```text
├── index.html
├── css/
│   └── style.css
└── js/
    └── script.js
```

## ⚙️ Setup Instructions

### Prerequisites

- Google Chrome, Microsoft Edge, Mozilla Firefox, or Safari
- Internet connection (if Bootstrap is loaded via CDN)

### Installation

```bash
git clone <repository-url>
cd tools-subscription-tracker
```

Alternatively, download the project as a ZIP file and extract it.

### Run

Open `index.html` in your preferred web browser.

**Optional:** Run using VS Code Live Server for a better development experience.
